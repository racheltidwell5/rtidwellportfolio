How the Internet Works Notes

How the Internet Works

  • large network of computers that communicate together
  • began in 1960s and evolved into public infrastructure in the 1980s
  • can link 2 computers together through network or router
  • router connects to modem
  • modem can receive information from infrastructure
  • ISP= Internet Service Provider

What is HTTP?

  • used to load webpages using hypertext links
  • sending a request out for info to get returned back
  • requests like Google searches
  • HTTPS= has security involved
